These are fonts for use with the Arduino GLCD library for KS0108 style
LCD display.

CalBlk36 is a large heavy sans serif font, not unlike Arial Black
CalLite24 is a smaller light sans serif font similar to Arial.

Add the *.h files to the fonts folder in the GLCD libary.

The files with no extension are font files for use with
Font Creator - http://www.apetech.de/fontCreator
